# Task 3 – Functions & Code Reusability

This repository contains the solutions for Day 3 Assignment based on Python Functions and Code Reusability.

The objective of this task is to understand how functions help in writing reusable, modular, and maintainable code in Python.

---

## Topics Covered

* Function Definition and Scope
* Parameters and Arguments
* Return Values and Data Types
* Default Arguments
* Variable-Length Arguments (`*args`)
* Lambda Functions
* Recursion
* Local, Global, and Nonlocal Variables
* Code Reusability using Helper Functions

---

## Practice Assignments Included

1. BMI Calculator Function
2. EMI Calculator
3. Recursive Factorial Program
4. Fibonacci Series Generator
5. Prime Number Checker
6. List Operations using:

   * map()
   * filter()
   * reduce()
